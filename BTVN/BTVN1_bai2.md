- CÁC KIỂU DỮ LIỆU TRONG PYTHON:
    int: số nguyên
    float: số thực
    str: chuỗi kí tự
    bool: (true, false) kiểu logic đúng sai
    list: danh sách có thể chứ nhiều kiểu
    tuple: bộ gtri cố định không thay đổi

- CÁC MỆNH ĐỀ ĐIỀU KIỆN VÀ VÒNG LẶP:
    1. Câu lệnh điều kiện (if - else):
        x = 10
        if x > 0:
            print("x là số dương")
        elif x == 0:
            print("x bằng 0")
        else:
            print("x là số âm")
    2. Vòng lặp for:
        for i in range(5):
            print(i)
    3. Vòng lặp while:
        x = 0
        while x < 5:
            print(x)
            x += 1
